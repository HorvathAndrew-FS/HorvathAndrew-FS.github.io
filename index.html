About Me

Welcome! My name is Andrew and I really enjoy code and developing objects, components and elements that reside on the internet. My path to the web began back in 2006 when I went out on a limb and enrolled in an Intro to Graphic and Web Design class. To my surprise, I became enamoured with HTML and CSS. I ventured off into the culinary world post graduation as I struggled to find employment in the Web Community, but I never stopped loving code and desired to do more with it.

I am now graduating from Full Sail University with a Bachelors in Web Developemnt. I have had the opportunity to learn from some of the greatest contributors in the Web Community and cannot wait to see where my journey into Web Development will take me!
